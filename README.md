# soigneeee

## Instructions
1. Clone the repo
2. Open in vscode or whatever IDE
3. Open terminal (check current directory is root folder **soigneeee**)
4. Install all required packages with:
```
npm install
npm update
```
5. Start the app with:
```
npm start
```

## Dependencies Used:
- React
- React-Scripts
- React-Dom
- React-Router-Dom
- Typeface-Roboto
- Material-Ui
- Formiz
- Mongoose
- MongoDB
- Axios
- Bcrypt
- Cors
- Concurrently
- Express
